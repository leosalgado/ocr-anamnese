# ocr-anamnese

### Dependencies:
- **Python 3.13**
- **uv**
- **Ollama**
  - **Model: llama3**

### 1. Clone

```bash
git clone https://github.com/leosalgado/ocr-anamnese.git && cd ocr-anamnese
```

### 2. Install/Sync dependencies
```bash
uv sync
```
### 3. Run
```bash
uv run src/main.py "path_to_anamnese"
```
